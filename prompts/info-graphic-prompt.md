Create an infographic using Dieter Rams / Braun-era minimalist design principles.

**Core Aesthetic:**
- "As little design as possible" — every element must serve a function
- Background: Braun White / Cream (#F0EDE5) — warm off-white, not sterile white
- Text: Black (#000000) for primary content, Grey (#8A8A87) for secondary/labels
- Single accent color: Signal Red (#C02820), used with extreme restraint

**Typography:**
- Clean, neutral sans-serif typeface (e.g., Helvetica, Inter, or similar)
- Clear visual hierarchy through weight and size, not decoration
- Body text: left-aligned, never centered
- Generous line spacing for readability

**Layout & Structure:**
- Grid-based composition with generous white space
- Thin, precise lines for dividers and structure — no decorative borders
- Functional visual flow: guide the eye logically through content
- Avoid shadows, gradients, textures, or illustrative elements

**Accent Color Usage (Red #C02820):**
- Use red ONLY for subtle emphasis on critical interactive or sequential elements
- Preferred treatment: red stroke/outline on circles around numbers (not filled)
- Circle stroke width: 2-3px, clean and precise
- Never use red for large areas, backgrounds, or decorative markers
- Maximum 3–5 red accents per infographic to maintain restraint

**Content Treatment:**
- Prioritize clarity over cleverness
- Use concise, scannable phrasing
- Tables and comparisons should use simple rules, no zebra striping
- Icons only if universally understood and functionally necessary

**Footer/Attribution:**
- Small, unobtrusive, grey text (#8A8A87) if needed
- Never compete with primary content

**Do NOT include:**
- Gradients, drop shadows, or glossy effects
- Decorative icons, illustrations, or clipart
- Corner markers, flourishes, or ornamental borders
- Centered body text or justified alignment
- More than one accent color
- Pure white (#FFFFFF) backgrounds — always use Braun cream (#F0EDE5)

**Color Palette Reference:**
- Braun White (background): #F0EDE5
- Braun Grey (secondary text): #8A8A87
- Signal Red (accent): #C02820
- Black (primary text): #000000

**Goal:** The design should feel calm, authoritative, and effortless to read — like a well-engineered Braun product from the Dieter Rams era.
